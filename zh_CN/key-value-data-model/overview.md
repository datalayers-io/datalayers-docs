# Key-Value 键值存储概述

Datalayers 键值存储服务是一个兼容 Redis 协议的分布式高性能键值存储系统。你可以继续使用现有的 Redis 客户端、工具链和访问方式，在较低迁移成本下获得更强的横向扩展能力与海量数据承载能力。

## 核心特性

- **协议 & 生态无缝兼容**  

    完全兼容 Redis 协议，支持通过标准 Redis 客户端、命令行工具及生态工具直接接入，业务代码零改造。

- **丰富的数据结构支持**  

    原生支持 Redis 常用数据结构，详见：[Redis 兼容性](./redis-compatibility.md)。

- **低成本高性能存储**

    支持 TB 至 PB 规模的 key-value 数据存储，依托分布式架构实现弹性扩展，降低资源成本与运维复杂度。

- **平滑迁移体验**  

    提供“零代码修改”的 Redis 迁移方案，业务无需调整逻辑即可从原生 Redis 或其他兼容服务无缝迁移至 Datalayers，快速完成存储底座升级。

## 注意事项

- 通过该协议连接仅支持 key-value 相关操作
- 目前 key-value 存储的数据暂不能通过 SQL 互通
- key-value 存储能力仅在集群模式下支持
