# MCP

## 配置

```toml
# The configurations of the MCP (Model Context Protocol) server.
[mcp]
# Whether to enable SSE.
# Default: true.
enable_sse = false

# Whether to enable SSE OAuth.
# Default: false.
enable_sse_oauth = false
```
