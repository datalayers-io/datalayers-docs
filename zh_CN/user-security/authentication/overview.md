# Datalayers 连接认证概述

Datalayers 提供 [静态认证](./static.md) 与 [RBAC](../rbac/overview.md) 两种连接认证方式，分别适用于不同的部署规模、权限模型和安全要求。

## 认证方式概览

下面是两种认证方式的核心差异，可根据环境复杂度和权限粒度要求进行选择：

| 特性 | 静态认证 | RBAC |
| --- | --- | --- |
| 认证机制 | 配置文件预置凭证 | 基于角色的访问控制 |
| 权限粒度 | 所有权限 | 细粒度权限控制 |
| 管理复杂度 | 简单 | 中等 |
