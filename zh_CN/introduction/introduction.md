# 产品概览

Datalayers 是一款面向工业物联网、车联网、能源等行业的**多模数据库产品**。得益于存算分离架构，在统一存储的基础上支持多种数据模型，如：时序存储、Key-Value、Queue、Stream、Docs等，为用户提供一站式的数据库服务平台。

## 功能特性

* **时序存储**
时序数据存储与处理是核心能力之一，具备高效写入、高压缩比、丰富的函数及海量时间线支持等特性。
* **自适应压缩算法**
根据数据类型自动选择最优压缩算法，实现数据高效压缩，降低存储成本。
* **数据缓存**
基于 LRU 策略，将热点数据加载至内存，以加速查询响应。
* **存算分离**
采用存储层与计算层分离的架构，支持存储与计算资源独立扩展和优化，各自独立进行弹性伸缩。
* **集群部署**
计算节点与存储节点均支持集群化部署，可按需独立进行弹性伸缩。
* **原生多模**
支持多种数据模型，包括时序存储、KV 存储等。用户可根据数据类型和访问模式选择最适合的数据模型，无需引入多套数据库系统。
* **原生 SQL 支持**
系统使用 SQL 作为主要交互语言，极大降低学习与迁移成本。
* **备份与恢复**
支持全量与增量备份，可将数据备份至 S3 等低成本对象存储。
* **边云同步**
提供边云同步功能，支持边缘侧数据向云端无缝同步。用户可在边缘实现自治，同时将数据汇聚到云端进行集中管理与分析。
